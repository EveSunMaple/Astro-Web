## 前言

你还在为爆 long long 而烦恼吗？

你还在为不会高精度而痛苦吗？

快使用`__int128`吧~

## 使用

因为\_\_int128 不支持标准的输入输出，我们只能用快读快输（还是个优点？）。

```cpp
inline __int128 read()
{
    __int128 x = 0, f = 1;
    char ch = getchar();
    while(ch < '0' || ch > '9')
    {
        if(ch == '-')
            f = -1;
        ch = getchar();
    }
    while(ch >= '0' && ch <= '9')
    {
        x = x * 10 + ch - '0';
        ch = getchar();
    }
    return x * f;
}
inline void print(__int128 x)
{
    if(x < 0)
    {
        putchar('-');
        x = -x;
    }
    if(x > 9)
        print(x / 10);
    putchar(x % 10 + '0');
}
```
